# Lottie 애니메이션 파일 위치

## 파일 경로 안내

| 용도 | 파일 경로 |
|---|---|
| 랜딩 히어로 애니메이션 | `public/assets/lottie/landing-hero.json` |
| 소개 페이지 캐릭터 애니메이션 | `public/assets/lottie/intro-character.json` |

## 사용 방법

1. 제작한 Lottie JSON 파일을 위 경로에 동일한 파일명으로 저장합니다.
2. `mp4` 영상을 사용하는 경우: `public/assets/videos/landing-hero.mp4` 에 저장합니다.
3. 코드 변경 없이 자동으로 적용됩니다.

## 스크롤 패럴랙스 사진

`public/assets/images/scroll-photos/` 폴더에 아래 이름으로 저장합니다:
- `photo-01.jpg`
- `photo-02.jpg`
- `photo-03.jpg`
- (추가 가능 — `LandingPage` 컴포넌트의 `SCROLL_PHOTOS` 배열에 경로 추가)
